# Android 真机验收清单

## 准备

- 使用 Android 8、10、12、14、15 或 16 中至少两台真实设备测试。
- 准备另一台手机发送普通测试短信，不使用真实银行卡或登录验证码测试。
- 后端未完成时可以只验证本地“等待上传”和“已过滤”计数。

## 首次启动和权限

- [ ] 首次启动不会自动弹出短信权限，也不会自动扫描。
- [ ] 点击“同意并授权读取短信”后才出现系统权限弹窗。
- [ ] 拒绝权限后页面保持可用，且没有读取短信。
- [ ] 再次允许权限后可以开始历史扫描。

## 历史短信

- [ ] 扫描后“等待上传 + 已过滤”的增量与手机已有收件/发件短信相符。
- [ ] 重复点击扫描不会重复增加同一条短信。

## 密码查看全部短信

- [ ] 首页存在“查看全部短信”入口。
- [ ] 输入错误密码、空密码或带空格的密码时提示“密码错误”，且没有读取短信。
- [ ] 输入固定密码 `88888888` 后显示本机系统短信库中的全部记录。
- [ ] 列表按时间从新到旧，并显示号码、完整正文、类型、已读状态、短信 ID 和可用 SIM 信息。
- [ ] 收件、已发送、草稿、发件箱、失败或排队短信存在时，方向标签显示正确。
- [ ] 命中上传黑名单的短信仍能在本机“全部短信”列表查看。
- [ ] 从查看页返回、切换到后台或锁屏后重新进入，10 分钟内保持解锁；到期或点击“锁定”后必须重新输入密码。
- [ ] 撤销 `READ_SMS` 权限后，正确密码只能进入授权提示，不能显示旧的页面正文。
- [ ] 草稿、失败发送和非 SMS 数据不会进入队列。

## 黑名单

- [ ] 发件人包含“淘宝”或“拼多多”的短信计入“已过滤”。
- [ ] 正文包含“退订”“促销”或“优惠券”的短信计入“已过滤”。
- [ ] 关闭某条规则后，新收到的匹配短信可以进入待上传队列。
- [ ] 新增、删除规则后，退出再进入页面仍能保留设置。
- [ ] 命中黑名单的短信不会出现在上传请求中。

## 新短信和进程回收

- [ ] App 在前台时收到普通短信，等待上传数量增加。
- [ ] 从最近任务划掉 App 后收到短信，Receiver 仍能建立队列记录。
- [ ] 可使用 `adb shell am kill com.myfile.smsbackup` 模拟普通进程回收，随后收到短信仍能建立队列记录。
- [ ] 使用系统“强行停止”后不会自行恢复；再次手动打开 App 后恢复任务。这是预期行为。

## 网络和服务器

- [ ] 未配置服务器时，短信保留在等待上传队列。
- [ ] 配置错误地址时“测试连接”明确失败，短信不丢失。
- [ ] HTTP 地址只有开启“允许 HTTP”后才能保存和测试。
- [ ] 恢复正确服务器和网络后，WorkManager 自动重试。
- [ ] 后端返回 `2xx` 后记录从等待上传变为已备份。
- [ ] 后端返回 `5xx` 或断网时保持待上传，并使用退避重试。
- [ ] 同一 `recordId` 重复发送时，后端只写一次。

## 重启和补漏

- [ ] 手机重启后无需打开页面，已安排的 WorkManager 任务仍保留。
- [ ] 重启后网络恢复，待上传记录继续上传。
- [ ] 每日补漏扫描不会产生重复记录。

## 位置轨迹和后台运行

- [ ] 只有在“设备与轨迹”页主动点击开始后才请求位置和通知权限。
- [ ] Android 12 及以上必须从可见页面启动位置前台服务，通知栏持续显示“正在记录位置”和“停止记录”。
- [ ] 首个有效定位点立即保存，之后相邻保存点的采集时间至少间隔 3 分钟。
- [ ] GPS 可用时可获得精确位置；仅允许大致位置或 GPS 不可用时能回退到网络定位。
- [ ] 页面退出、锁屏和从最近任务划掉后通知仍存在，继续产生轨迹点。
- [ ] 点击通知“停止记录”或页面“停止记录”后不再新增轨迹点，并结束当前轨迹会话。
- [ ] 轨迹分析过滤精度大于 120 米、速度大于 80 米/秒和不足 5 米的漂移点。
- [ ] 电池白名单和应用详情按钮能打开系统设置；华为、小米、OPPO、vivo、三星设备分别验证允许后台活动/自启动。
- [ ] 系统“强行停止”、重启或厂商深度冻结后不会承诺自行恢复；重新打开应用并主动启动轨迹是预期流程。

## 联系人和设备状态

- [ ] 未授予联系人权限时不读取联系人，并显示单独授权按钮。
- [ ] 授权后显示所有含手机号的联系人，支持按姓名或号码搜索，重复号码不会重复显示。
- [ ] 联系人、位置坐标、设备状态只在 10 分钟查看会话内显示，点击“锁定”后立即清空页面数据。
- [ ] 设备页正确显示厂商、型号、Android/API、电量、充电状态、健康度、温度、电压、内存、存储、网络和电池优化状态。
- [ ] 所有新增数据只保存在本机；没有新增联系人、坐标或设备诊断上传请求。

## 隐私检查

- [ ] 应用图标和同步状态始终可见。
- [ ] 页面提供同步开关和服务器地址。
- [ ] Logcat、HBuilderX 控制台和普通日志中没有短信正文。
- [ ] Logcat、HBuilderX 控制台和普通日志中没有联系人、手机号或经纬度。
- [ ] 手机持有人可以在系统设置中撤销短信权限或卸载应用。
