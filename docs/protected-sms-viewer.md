# 受保护短信与图片浏览

在首页选择“查看全部短信”，输入查看密码后即可进入受保护浏览器。

- 会话从首次解锁开始固定有效 10 分钟；刷新、返回首页、进入聊天详情和切到后台都不会续期。
- 点击“锁定”或到期后，应用会清除内存中的密码、短信、彩信附件和相册索引；重新查看必须再次输入密码。
- “会话”按 Android 系统短信线程或号码显示与谁聊天；点击可查看所有收到与已发送的完整短信、时间和 MMS 图片附件。
- “全部”“收到”“已发送”可直接查看相应类别的完整正文。
- “图片”页包含 MMS 图片。首次查看本机相册时，应用会单独请求图片读取权限：Android 13 及以上使用 `READ_MEDIA_IMAGES`，Android 8–12 使用 `READ_EXTERNAL_STORAGE`。
- 应用不会请求通讯录权限，因此会话名称显示系统短信库提供的号码或地址；不会复制或上传相册照片。

固定查看密码仅用于防止普通误看，不能替代设备锁、真实用户认证或加密存储。
